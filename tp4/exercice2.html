Question 1 : Sélecteur de groupe

Comment cibler à la fois tous les titres principaux, tous les éléments de la classe "fleurs", ainsi que l'élément dont l'identifiant est "cadre" ?

h1, h2, h3, .fleurs, #cadre { /* Styles appliqués à tous ces éléments */ }

Question 2 : Sélecteur descendant

Comment cibler uniquement les éléments <p> contenus au sein d'un élément <section> ?

section p { /* Styles appliqués uniquement aux paragraphes dans une section */ }

Question 3 : Sélecteur :nth-child()

Colorer le texte du troisième paragraphe en rouge sans modifier le code HTML.

p:nth-child(3) { color: red; }

Ou le sélecteur :nth-of-type() :

p:nth-of-type(3) { color: red; }

Question 4 : Sélecteur adjacent

Écrire le code HTML correspondant à la règle CSS suivante : p + hi.kivi + p strong

<p>Premier paragraphe</p> <hi class="kivi">Élément hi avec classe kivi</hi> <p>Deuxième paragraphe <strong>texte en gras ciblé</strong></p>

Question 5 : Sélecteur d'attribut

Que signifie la syntaxe suivante ? img[alt*="logo"]

Ce sélecteur cible toutes les images (<img>) dont l'attribut "alt" contient la sous-chaîne "logo".

Exemples :

<img src="im1.png" alt="Mon logo"> /* Ciblé */ <img src="im1.jpg" alt="logo de l'entreprise"> /* Ciblé */ <img src="im1.jpg" alt="photo"> /* Non ciblé */
